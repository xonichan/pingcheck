# Contributing to Pingcheck

Спасибо за ваш интерес к проекту Pingcheck! Эта документация поможет вам внести свой вклад.

## Как внести вклад

### 1. Создание Issue

Если вы нашли баг или у вас есть идея для улучшения:
- Проверьте существующие issues перед созданием нового
- Используйте шаблон issue при создании
- Опишите проблему максимально подробно

### 2. Pull Request

При создании pull request:
- Следуйте стилю кода проекта
- Добавьте комментарии для сложной логики
- Обновите документацию при необходимости
- Укажите номер issue (если есть)

### 3. Стиль кода

- Используйте Python 3.8+
- Соблюдайте PEP 8 (где возможно)
- Используйте type hints для функций
- Добавляйте docstrings для новых функций

### 4. Тестирование

- Проверяйте работу на разных ОС (Windows, Linux, macOS)
- Убедитесь, что все зависимости установлены корректно

### 5. Связь

- Вопросы можно задавать в issues
- Для срочных вопросов используйте email из README

## Что мы принимаем

- Исправление багов
- Улучшение документации
- Новые функции (обсуждаем в issue)
- Оптимизация кода

## Благодарности

Каждый вклад ценится! Мы благодарим всех контрибьюторов.
